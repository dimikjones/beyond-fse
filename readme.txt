== Description ==

Beyond FSE is a modern, lightweight Full Site Editing theme built for bloggers who value speed, clean aesthetics, and complete design flexibility. Originally crafted to power my personal blog, it has been fully refined and optimized to give back to the WordPress community. 

Built from the ground up with a performance-first mindset, Beyond FSE ensures exceptional performance and a minimal footprint. It gives you total control over your site's layout using the native block editor, making website customization smooth and intuitive.

Key Features:
* Full Site Editing (FSE): Effortlessly customize your headers, footers, and page templates.
* 4 Style Variations: Includes the polished default look plus three distinct alternative styles to instantly change your site's vibe.
* 2 Custom Page Patterns: Pre-designed block layouts to help you launch content in seconds.
* 4 block styles: core/list (Check Circle), core/code (Invert), core/separator (Stripes, Thin, Thin Wide), core/preformatted (Invert)
* Highly Optimized: Clean, standard-compliant code optimized for excellent core web vitals and SEO structure.

== Installation ==

= Minimum Requirements =

* WordPress 6.9 or greater
* PHP version 7.4 or greater

= Installation =

From your WordPress dashboard

1. Go to your WordPress Dashboard -> Themes -> Add New
2. Upload theme .zip file
3. Activate Beyond FSE theme from your Themes page.
4. Enjoy :)
