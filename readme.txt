== Installation ==

= Minimum Requirements =

* WordPress 6.9 or greater
* PHP version 7.4 or greater

= Installation =

From your WordPress dashboard

1. Go to your WordPress Dashboard -> Themes -> Add New
2. Upload downloaded .zip file
3. Activate Beyond FSE theme from your Themes page.
4. Enjoy :)
